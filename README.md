## Edit

Everything is in `index.html`. Content is inline, styling is inline. One file, no build step. Open it in VS Code, edit, commit, push. GitHub Pages redeploys within a minute.

## Custom domain (optional)

If you want this at something like `guide.tolaniai.com`:

1. Add a `CNAME` file in the root containing just your domain (no protocol)
2. In your DNS provider, add a CNAME record pointing your subdomain to `tolaniomitokun.github.io`
3. In GitHub Settings → Pages → Custom domain, enter your domain and check "Enforce HTTPS"
